Password Cracking

Your passwords in website for example facebook is saved in MD5 Hash format like for example if you keep your password as 123 then your MD5 hash would be 202cb962ac59075b964b07152d234b70 and it would be much easier to crack small passwords that's why you are recommended to keep your password unique and big so that the hash generated is not common and it should be hard for the hacker to crack the hash. As if you try to enter your hash in the password area then it would identify it as incorrect so hackers need to crack the hash to get access.

There are 3 types of password attacks-

  • Dictionary Attack- When a hacker attempts to get the password by many sets of word until he finds the correct set of words as password.
  • Brute force attack- In which hacker submits many password until he founds one of them as correct.
  • Keyloggers- In which hacker has control to your keyboard and can monitor the activity when you are using your keyboard to logging in to your account.

In password cracking python can be used for making tools which can crack the password like Dictionary attack, brute force attack tools can be made by python.
