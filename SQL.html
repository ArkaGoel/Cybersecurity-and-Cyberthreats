SQL Injection

A type of cyber attack in which a hacker uses a piece of SQL(Structured Query Language - a programming language designed to get information out of and put it into a relational database) code to manipulate a database and gain access to potentially valuable information.

It is basically retrieving hidden data which like passwords from the database. By the Structured Oriented Language.

It Occurs when an uninted data enters a program from untrusted source.

The main consquences for this are-

  1. Confidentiality(Secret) - Since SQL database hold very sensitive data , loss of Confidentiality is a frequent problem.
  2. Authentication(Recognizing a user's identify)- is poor SQL commands are used to check user names and passwords, then it may be be easy to enter to someone else's account without knowing password.
  3. Rights- Can change the information and rights.
