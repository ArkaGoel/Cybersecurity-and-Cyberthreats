Types of Security-

Critical infrastructure security- Helps in protecting the computer systems, networks and other assets that society relies upon for national security, economic health and for public safety. National Institute of Standards and Techonology(NIST) has created a cybersecurity framework to help organizations in this area, while the U.S department of homeland security(DHS) provides additional guidance.

Network security- These are the security measures taken to avoid the intruders/hackers which can protect from them from hacking into wired and wireless networks.

Application security - Hackers can hack into applications both which are on premises and cloud. To avoid that security measures are taken while the application is in the development.

Cloud security - It is a security to protect clod based data such as applications.

Information security -A data security measure such as the Genenral Data Protection Regulation (GDPR) which helps to prevent your very sensitive data from an unauthorized access

End-user education - Building a security awareness across the organizations to strengthen the security. Like to train the users about deleting the suspicious emails and etx.

Disaster recovery / business continuity planning - tools and procedures for responding to unplanned events like natural disasters, power outages or cybersecurity incidents.
